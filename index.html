📖 How to Manage Categories, Levels & Questions

1

Understanding the File Structure

Open kannada-kotyadi-quiz.html in any text editor (VS Code recommended). Search for the section marked // QUESTION BANK. All questions are stored in the QUESTIONS object, organized by category ID and level number.

2

Adding a New Category

Find the CATEGORIES array and add a new entry:

  • id: unique identifier e.g. 'geography'
  • label: object with en, kn, hi translations
  • icon: any emoji e.g. '🗺️'

Then add a matching entry in the QUESTIONS object with 10 level arrays, each containing 10 questions.

3

Adding Questions to a Level

Each question uses the Q() helper function with this format:

  • Arguments: Q(english_question, english_options[], kannada_q, kannada_opts[], hindi_q, hindi_opts[], correct_answer_index, difficulty)
  • correct_answer_index: 0 = A, 1 = B, 2 = C, 3 = D
  • difficulty: 0 = Easy, 1 = Medium, 2 = Hard

Recommended difficulty: Levels 1–3 → 0 (Easy), Levels 4–6 → 1 (Medium), Levels 7–10 → 2 (Hard)

4

Changing Number of Levels (5–10)

To reduce levels from 10 to 5, simply provide only 5 arrays in that category's question bank. The level sidebar auto-generates based on how many level arrays exist. Also update the PRIZES array accordingly at the top of the script.

5

Setting Up Firebase (for Participant Tracking)

  • Go to console.firebase.google.com → Create a new project
  • Add a Web App → copy the firebaseConfig object
  • Enable Firestore Database → Start in test mode
  • Paste the config into the firebaseConfig block at the top of this file
  • Participant data is saved to the quiz_participants collection automatically
6

Deploying to GitHub Pages

  • Push the HTML file to your GitHub repository
  • Go to Repository Settings → Pages → Set Source to main branch / root
  • Your quiz will be live at https://username.github.io/repo-name/
  • Share via WhatsApp or Instagram as a direct link
7

Changing Pass Threshold

Find const PASS_THRESHOLD = 0.9; near the top of the script section. Change 0.9 to any value between 0 and 1 (e.g. 0.8 for 80%, 0.7 for 70%).

8

Adding Prize Amounts

Find the PRIZES array at the top of the script. Each entry corresponds to a level (Level 1 = index 0). You can customize the amounts freely, e.g. '₹5,000', '₹25,000' etc.
